Dunham Code Challenge

Objective:

Edit a small React app to add 2 pages:

  1. A form to update a User object
  2. A "thank you for submitting" page.
Only the form page should be accessible in the navigation menu.

Your form should:

  • get the User you want to update
  • display the user information and then
  • send the updated User back to the server.

Your app will do this by interfacing with a poorly documented API, which has been "detailed" below.

You can get started by forking this pen so that you can edit it freely.

This pen has Bootstrap V5.0 and React.js V17.0.2 installed.

API Reference

The API is located at https://manage.thedunhamgroup.com/aws-server/

Endpoints

List - GET - Lists all API endpoints

GetUser - GET - returns a single User object

SetUser - POST - updates an existing User object

User Object

UserID - INT - The user's ID

Email - STRING - The user's email - must be a valid email address

ProfilePhoto - STRING - The user's ProfilePhoto

Name - STRING - The user's Name - must be in the form "FIRST LAST"

Birthday - The user's date of birth
